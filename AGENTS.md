# MCP Servers Configurados

## Context7 (remote)
Busca documentacao atualizada de APIs e bibliotecas.
Use: `use context7` para pesquisar documentacao.

## Grep by Vercel (remote)
Busca exemplos de codigo no GitHub.
Use: `use gh_grep` para encontrar exemplos de implementacao.

## Excalidraw (remote)
Cria diagramas e ilustracoes visuais (design).
Use: `use excalidraw` para criar diagramas.

## Sequential Thinking (local)
Resolucao estruturada de problemas passo a passo.
Use: `use sequential_thinking` para problemas complexos.

## Fetch (local)
Busca conteudo web para documentacao e referencias.
Use: `use fetch` para acessar paginas web.

## Skool (local)
Gerenciamento de comunidade Skool.com.
Use: `use skool` para interagir com Skool.
Atencao: Configure o cookie `auth_token` em `~/.config/skool-mcp/config.json`
